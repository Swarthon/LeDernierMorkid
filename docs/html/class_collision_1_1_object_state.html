Detailed Description

Class defines the way Collision and Graphics are synchronized.

ObjectState derives from btMotionState and overloads the functions to synchronize Ogre::SceneNode and btRigidBody from a Common::GameEntity. To use it, do as follow

rigidBody->setMotionState(new Collision::ObjectState(transform, gameEntity, logicSystem));

10 It aims to provide a full open source game which would be free.

24 ## Requirements
25 
26 You will need a C++ compiler, see [Compiling](#compiling) section for more informations. The project need the [OGRE library](http://ogre3d.org), which itself needs the [SDL2 library](http://libsdl.org/). To compile them, search on their website or use prebuilt files.
27 
28 It has already been tested on Linux and Windows. I highly recommend to use the GNU Compiler (MinGW on Windows), with wich it has already been compiled.
29 
30 ## Dependencies
31 
32 As written below in [Requirements](#requirement), this project has a few dependencies :
33  - **Ogre3D**, version 2.1, the components you need are HlmsPbs and HlmsUnlit. The version 2.1 is not released yet but you can download it by doing `hg clone https://bitbucket.org/sinbad/ogre && cd ogre/ && hg pull && hg update v2-1`, assuming you have Mercurial.
34  - **SDL2**, the library used by Ogre to get the input. This library is very common and you can download it either from you package manager either from [the official site](http://libsdl.org/)
35  - **CEGUI**, version 0.8, a library which provide Ogre3D support for GUI. The version 0.8, which is yet only doesn't compile for Ogre 2.1, so I use a patch found [here](http://www.ogre3d.org/forums/viewtopic.php?f=25&t=82911&start=25), which gives [this link](https://pastebin.com/BB7CmUtw). To download the sources, the patch and apply it. `hg clone https://bitbucket.org/cegui/cegui && cd cegui/ && hg pull && hg update v0-8 && wget https://pastebin.com/raw/BB7CmUtw -O cegui.patch && patch -p1 < cegui.patch`, assuming you have Mercurial, wget and patch.

37 ## Compiling
38 
39 Before trying to compile the project itself, check if you have all the dependencies, see [Dependencies](#dependencies). To compile this code, you will need to run cmake. Use either `cmake-gui` or `cmake` to configure the project and then compile it. If you have trouble in doing it, just contact me, I will try to answer you as fast as possible, for more information see [Contacting](#contacting).

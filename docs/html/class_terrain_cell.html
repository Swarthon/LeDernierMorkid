bool TerrainCell::merge ( TerrainCell *  next)

Merges another TerrainCell into 'this' for reducing batch counts. e.g. Two 32x32 cells will merge into one 64x32 or 32x64 Two 64x32 cells will merge into one 64x64 A 32x64 cell cannot merge with a 32x32 one. A 64x32 cell cannot merge with a 32x32 one.

Remarks
Merge will only happen if the cells are of the same LOD level and are contiguous.
Parameters
nextThe other TerrainCell to merge with.
Returns
False if couldn't merge, true on success.

Queues message 'msg' to be sent to a destination MessageQueueSystem. This function must be called from the thread that owns 'this' The 'dstSystem' may live in any other thread.

Parameters
dstSystemThe MessageQueueSystem we want to send a message to.
messageIdThe type of message sended
msgThe message itself. Structure must be POD.
Remarks
The message is not instantely delivered. It will be sent when flushQueuedMessages gets called.

void Common::SdlInputHandler::setMouseRelative ( bool  relative)

Set the mouse to relative positioning mode (when not supported by hardware, we emulate the behavior). From the SDL docs: While the mouse is in relative mode, the cursor is hidden, and the driver will try to report continuous motion in the current window. Only relative motion events will be delivered, the mouse position will not change

Method to add a resource location to for a given resource group.

Parameters
archNameThe name of the resource location; probably a directory, zip file, URL etc.
typeNameThe codename for the resource type, which must correspond to the Archive factory which is providing the implementation.
secNameThe name of the resource group for which this location is to apply. ResourceGroupManager::DEFAULT_RESOURCE_GROUP_NAME is the default group which always exists, and can be used for resources which are unlikely to be unloaded until application shutdown. Otherwise it must be the name of a group; if it has not already been created with createResourceGroup then it is created automatically.
Remarks
This method only call Ogre::ResourceGroupManager::getSingleton().addResourceLocation

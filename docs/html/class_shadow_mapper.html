Gets how many steps are needed in Bresenham's algorithm to reach certain height, given its dx / dy ratio where: dx = abs( x1 - x0 ); dy = abs( y1 - y0 ); and Bresenham is drawn in ranges [x0; x1) and [y0; y1)

Parameters
yHeight to reach
fStep(dx * 0.5f) / dy;
Returns
Number of X iterations needed to reach the the pixel at height 'y' The returned value is at position (retVal; y) which means (retVal-1; y-1) is true unless y = 0;
